# MediaIndex 0.4.6

## 企业微信图文回执

- 企业微信资源任务完成后优先发送带海报的图文消息。
- 海报先下载到 MediaIndex 持久化缓存，再由 MediaIndex 自己的公开只读接口提供。
- 海报缓存仅允许 TMDB 官方图片域名，校验 JPEG/PNG 格式并限制单图最大 8 MB。
- 相同海报直接复用缓存，超过 90 天的旧缓存会自动清理。
- 海报缓存、公开地址或企业微信图文发送不可用时自动回退为文字消息。
- 图文卡片点击后进入对应的智能追更、愿望单或待确认页面。
