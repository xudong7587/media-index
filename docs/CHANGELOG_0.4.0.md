# MediaIndex 0.4.0

## 新增

- 新增站内通知中心，支持未读数量、全部与未读筛选、逐条已读、全部已读和清空。
- 通知支持跳转到智能追更、愿望单、待确认和设置页面。
- 新增通知持久化接口和转存任务终态事件同步。
- 覆盖转存完成、任务已提交、需要人工确认、暂无资源和处理失败等事件。

## 改进

- 通知事件使用稳定业务键去重，避免轮询产生重复消息。
- 已撤销和已替代的审核任务不会生成无效失败通知。
- 通知面板适配桌面、移动端、浅色和深色模式。
- 补齐通知加载、空数据、失败重试和键盘焦点状态。

## 验证

- 前端 TypeScript 检查和 Vite 生产构建通过。
- 后端共 118 个自动化测试通过，其中包含 2 个通知中心测试。
- 已完成桌面端、390px 移动端、浅色和深色模式浏览器检查。
