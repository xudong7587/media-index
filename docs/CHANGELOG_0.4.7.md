# MediaIndex 0.4.7

## 企业微信编号交互

- 资源名称匹配到多个影视条目时不再自动选择第一项。
- 按相关度列出最多 5 个候选，展示电影、剧集或综艺类型及年份。
- 用户回复数字后才创建转存任务，发送“取消”或 `/cancel` 可终止选择。
- 编号选择按企业微信 UserID 独立保存，有效期 30 分钟，服务重启后仍可继续。
- 转存或智能追更进入待确认时，直接列出资源候选并支持回复数字确认。
- `/review` 支持先选择待确认任务，再选择该任务的资源候选。
- 候选确认复用网页端的安全确认和重新匹配逻辑。
- 转存任务保存显示标题，通知和待确认列表不再退化为任务编号。

## 设置结构

- 顶部导航移除独立推送入口。
- 设置页增加“基础设置”和“通知设置”两个子页签，通知设置位于右侧。
- 企业微信、Telegram、消息回调和测试功能统一放入通知设置。
- 新增两列式“内置指令速查”，移动端自动切换为单列。
