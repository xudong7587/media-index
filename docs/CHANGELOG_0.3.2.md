# MediaIndex 0.3.2

## 集数匹配优化

- 支持 `1168.mp4`、`第1168集`、`E1168`、`S23E1168` 等四位集号。
- 四位集号严格要求与 TMDB 目标集完全一致，不会把年份当成目标集。
- 支持四位数的连续集范围写法。
- 普通剧集与短篇动画的裸 `01、02、03` 只有在分享中形成至少三个连续视频文件时才升为高置信。
- 数字序列自动执行还要求 PanSou 候选标题明确匹配媒体；标题弱的合辑只进入确认，不自动转存。
- 综艺仍使用独立的期数、上中下、播出日期和衍生内容排除规则。

## 电影候选修复

- 包含搜索词的候选正文不能伪造片名命中。
- 无关日期合辑不会进入待确认。

## 文档

- README 只保留产品定位、部署方式、智能追更特点和版权免责声明。
- 版本变更统一维护在 GitHub Releases。

## 验证

- 《航海王》第 23 季真实验证：`1168 4KHDR.mp4` 正确映射为 `航海王.1999.S23E1168.mp4`。
- 《海洋奇缘：启航》无关候选真实验证为 `candidate_count=0`。
- 自动化测试增加到 88 项。
