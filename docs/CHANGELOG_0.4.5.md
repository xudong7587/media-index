# MediaIndex 0.4.5

## 企业微信资源指令

- 直接向企业微信自建应用发送影视资源名，自动匹配 TMDB 条目并默认保存到网盘。
- 发送“本地 资源名”或“本地：资源名”，自动改为保存到本地。
- 支持“网盘 资源名”和“云端 资源名”的显式写法。
- 电视剧与综艺自动选择当前最新的有效季度。
- 任务开始时回复匹配结果、保存目标和任务编号，完成后再次回复转存结果。
- 无可用资源时沿用现有流程自动加入愿望单，需要人工选择时进入待确认。
- 推送设置页和 `/help` 增加资源指令说明。

## 内部调整

- 提取可复用的转存任务入队函数，使网页和企业微信使用同一套任务及去重逻辑。
